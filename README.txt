## Puzzle Program

### Purpose
This program assembles a sequence of fragments into the longest possible single-row puzzle.

### Prerequisites
- JDK 8 or later.

### How to Run
1. If you want to use another file, place your file in the `src/` folder and change
 static final String FILE_NAME with your file name.

2. Compile the program from pazzzzl directory:
   ```bash
   javac src/Puzzle.java
   ```
3. Run the program:
   ```bash
   java src/Puzzle.java
   ```
